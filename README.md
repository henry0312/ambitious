# Ambitious

**Ambitious** adds advanced and experimental features to [Keras](https://github.com/fchollet/keras).

## Requirements

* Python 3.5 or later  
  (Note that I don't support Python 2.7)
* Keras 1.0 or later

## Install

```sh
make install
```
